# FUTURE_ML_02 - Support Ticket Classification

## Objective
Build a system to automatically classify customer 
support tickets and assign priority levels.

## Dataset
Customer Support Ticket Dataset

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
- TF-IDF Vectorizer

## Kaggle Notebook
[View Notebook](https://www.kaggle.com/code/moni35/notebook278e8c56a3)

## Results
- Classified tickets into 5 categories
- Assigned High/Medium/Low priority
- Created visualizations

## Priority Logic
- Technical issue → High
- Refund request → High
- Billing inquiry → Medium
- Cancellation request → Medium
- Product inquiry → Low
